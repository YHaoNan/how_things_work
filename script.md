# 矩阵乘法

> 线性代数太简单了，我已经学过7次了

就像上面所说，使用数值来理解线性代数是痛苦的、毫无启发的，并且会经常性的遗忘。就比如如果我跟你说，$A\times B = C$，$C_{ij}$是$A$的第$i$行和$B$的第$j$列的点积，你知道我在说什么吗？即使你通过大量习题练习，达到了这个计算的肌肉记忆，问题是下次用的时候你还记得吗？坦白讲，每次我需要用到矩阵乘法时，我都需要百度一遍让它重新进入我的脑子。

但是，线性代数中的操作是有几何意义的。

本期视频的目标有两个：
1. 从几何意义上理解矩阵乘向量，即形如$Ax=b$，它实际在干什么
2. 从几何意义上理解矩阵相乘，即形如$A \times B = C$，它实际在干什么，并且我们会推导出，为什么矩阵乘法是这样计算的

# 矩阵乘法推导

> 举例$AB$，观察原始单位向量$\hat{x}$在经历B的映射后变成了什么，然后它再经历A的映射，变成了什么。同理，再观察$\hat{y}$。
> 你会发现，这个过程的计算就是矩阵乘法
