# 棋盘算法
但对于玩家来说，小孩子才做选择，玩家往往统统都想要！如何在保证高分辨率的情况下，保持小球运动帧率在可接受范围内？如在2K分辨率下，也能保住30fps的帧率？

> 思维导图切换至棋盘算法

我们立刻就能想到一个取巧的办法，我们每帧只选取一半的像素点进行渲染，下一帧再渲染另一半的像素点，在快速交替的情况下，人眼可能难以察觉这一差异，但重要的是，计算量整整少了一半，我们可以以1080p的压力渲染2K视频了。这种经典的简单算法叫做棋盘算法，相比于我们今天要介绍的技术，它要简单朴素的多，但它却是一个非常好的讲解其它技术的基础。

看过了棋盘算法，我们便可以引入我们今天的主角：超分技术。它采用了另外的思路来解决帧率和分辨率不能同时拉高的矛盾。

# 超分技术简要介绍
超分技术全称是超分辨率技术，它的基本原理是：

1. 游戏引擎渲染一个低于目标分辨率的帧（如720P）
2. 它将这一帧与一些算法需要的数据打包交给超分算法，如运动向量，历史帧图片等信息
3. 超分算法将其渲染成更高分辨率的帧（如4K）

# DLSS
DLSS是英伟达出品的超分技术，它使用机器学习方案，在训练时学习大量高分辨率游戏图像，以达到提升分辨率的目的。由于运行模型对性能有一些损耗，为了实现高效的AI运算，英伟达从图灵架构（20系）开始的显卡，都在其流处理器中集成了名为‘Tensor Core’的张量核心。DLSS正是利用这些核心进行加速的。因此，DLSS是一个与英伟达硬件深度绑定的闭源技术，只有你购买了英伟达的支持DLSS的显卡，才可能使用它。

# FSR
DLSS的缺点在于它的通用性低，AMD的FSR解决了这一问题，它是基于规则的，人工设计的，低损耗的超分算法，算法本身是纯开源的软件，超分算法被打包成着色器程序丢到GPU上执行，所以它的通用性会强很多，几乎兼容全部现代GPU。

总的来说，DLSS凭借硬件和AI的优势，通常在画质和性能效率上表现最佳；而FSR则以无与伦比的开放性和兼容性取胜，让更多玩家能享受超分技术的好处，同时经过数代的优化，画质也能接近于DLSS。
